## Recursos Identificados

- **Películas (movies):** Representa las cintas que se proyectan en el cine. Contiene título, duración y clasificación.
- **Salas (rooms):** Espacios físicos donde se proyectan las películas. Contiene capacidad y tipo de pantalla.
- **Usuarios (users):** Clientes registrados en el sistema que pueden realizar reservas.
- **Reservas (reservations):** La acción de bloquear una butaca específica para un usuario en una sesión.
- **Pagos (payments):** Transacción económica asociada obligatoriamente a una reserva.
