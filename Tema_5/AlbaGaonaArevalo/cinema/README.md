# UNIR-CINEMA

This project is a simple web application that displays a list of movies, including their posters, synopses, and other details.

## About the project

The application consists of the following components:

- `Header`: The header of the page.
- `MovieList`: A component that displays a list of movies.
- `Movie`: A component that displays the details of a single movie.
- `Footer`: The footer of the page.

The main view of the application is `Movies`, which integrates all the above components to display the full page.
